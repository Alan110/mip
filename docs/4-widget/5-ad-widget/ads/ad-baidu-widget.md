# 类型: ad-baidu

网盟广告。

## 示例

在MIP HTMl中，示例如下：

```
<div class="mip-adbd">
    <mip-ad 
        type="ad-baidu" 
        cproid="u2697398">
    </mip-ad>
</div>
```
## 属性

组件所涉及的属性有：广告类型（type）和 网盟广告id（cproid)

- **广告类型（type）**
    
    - 是否必填：是

    - 说明：广告类型（ad-comm/通用广告，ad-baidu/网盟广告）

- ** 网盟广告id（cproid)**

    - 是否必填：是

    - 说明：网盟广告id。
