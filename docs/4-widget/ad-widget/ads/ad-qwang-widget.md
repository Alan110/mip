# ad-qwang

全网推荐

## 示例

``` 
 <mip-ad type="ad-qwang" cpro_psid="u2619809" cpro_pswidth="auto" cpro_psheight="230"></mip-ad>
```

## 属性

基础广告组件所涉及的属性有：类型（type）id标识符（cpro_psid），宽度（cpro_pswidth），高度（cpro_psheight)

- **类型(type)**
    
    - 是否必填：是

    - 说明：全网推荐type为ad-qwang


- ** id标识符(cpro_psid)**
    
    - 是否必填：是

    - 说明：投放id。

- **宽度(cpro_pswidth)**
    
    - 是否必填：否

    - 说明：不填默认为auto 

- **高度（cpro_psheight)**

    - 是否必填：否

    - 说明：不填默认为230
