## 1. 使用

**mip-recommend**组件提供统一样式的相关推荐和新闻热点模块，需求方只需要根据规范在dom结构中加入组件即可。代码示例：

```
    <mip-recommend>
        <div class="recommends">
            <div class="recommends-header">相关推荐</div>
        </div>
        <div class="hotpoint">
            <div class="hotpoint-header">新闻热点</div>
        </div>
    </mip-recommend>
```
